## Capital Placement Task 1

- clone repo
  
```shell
    git clone https://github.com/Tk-brainCodes/Capital-Placement-Task1.git
 ```

- cd into directory
- 
  ```shell
    cd capital-placement
  ```
- install dependencies
  
  ```shell
    npm install
  ```
- execute yaml script
  
  ```shell
  prism mock FormApplication.yaml
  ```
  - run the server
    
    ```shell
    npm run dev
    ```
